Fycons

Your friendly neighborhood icon-font library

Usage

Fycons are generally tracked and vertically aligned for you. When used in parent elements such as buttons, they should correctly display and scale with inherited font-sizes. The advised tag to use for fycons is <i class="fycon-*" aria-hidden="true"></i>.

The attribute aria-hidden="true" prevents some screenreaders from announcing the character code (reference).

Note: class names shown below under "List of Fycons" need to be prefixed with fycon-. For example, <i class="fycon-action-etc" aria-hidden="true"></i>.

With buttons

Add the fycon tag inside of a container element. Do not add the fycon class to container itself.

<button class="lf-btn lf-btn-primary">
  <i class="fycon-plus" aria-hidden="true"></i> New Collection
</button>

<button class="lf-btn lf-btn-default">
  <i class="fycon-action-like" aria-hidden="true"></i> Like
</button>

<button class="lf-btn lf-btn-link">
  <i class="fycon-action-reply" aria-hidden="true"></i> Reply
</button>

Colored Fycons

Social media fycons ship with colored variants. Simply add the .colored class and the correct brand color will be applied to it.

<i class="fycon-source-livefyre colored" aria-hidden="true"></i>
<i class="fycon-source-facebook colored" aria-hidden="true"></i>
<i class="fycon-source-twitter colored" aria-hidden="true"></i>
...
