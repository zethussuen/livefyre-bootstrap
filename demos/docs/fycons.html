Fycons

Your friendly neighborhood icon-font library

Usage

Fycons are generally tracked and vertically aligned for you. When used in parent elements such as buttons, they should correctly display and scale with inherited font-sizes. The advised tag to use for fycons is <i class="fycon-*"></i>.

Note: class names shown below need to be prefixed with fycon-. For example, <i class="fycon-action-etc"></i>.

With buttons

Add the fycon tag inside of a container element. Do not add the fycon class to container itself.

<button class="lf-btn lf-btn-primary">
  <i class="fycon-plus"></i> New Collection
</button>

<button class="lf-btn lf-btn-default">
  <i class="fycon-action-like"></i> Like
</button>

<button class="lf-btn lf-btn-link">
  <i class="fycon-action-reply"></i> Reply
</button>

Colored Fycons

Social media fycons ship with colored variants. Simply add the .colored class and the correct brand color will be applied to it.

<i class="fycon-source-livefyre colored"></i>
<i class="fycon-source-facebook colored"></i>
<i class="fycon-source-twitter colored"></i>
...
